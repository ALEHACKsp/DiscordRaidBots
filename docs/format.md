# Format for tokens and snowflakes

## `token.txt`

```txt
token1
token2
token3
token4
```

## `snowflake.txt`

```txt
userid1
userid2
userid3
userid4
```

## `config.json`

```json
{ "owner_snowflake":USERID HERE, "invite_code":"INVITE CODE HERE" }```


(If at any time the snowflake causes an error, just add quotes `"` around the user ID like so:

```json
{ "owner_snowflake":"USERID HERE", "invite_code":"INVITE CODE HERE" }```
